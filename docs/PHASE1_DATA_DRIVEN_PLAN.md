## 📊 КРИТЕРИИ КАЧЕСТВА

### Тег имеет смысл если:
- ✅ Встречается ≥30 раз в промтах (≥0.3% библиотеки)
- ✅ Можно однозначно классифицировать
- ✅ Не дублирует существующий с тем же смыслом

### Примеры:
```
✅ ТЕХНО: 1,234 треков ← популярно, добавляем
✅ МЕЛОДИК ТЕХНО: 456 треков ← достаточно, добавляем подтег
❌ АМАПИЯНО: 2 трека ← слишком мало, пропускаем
❌ PHONK: 0 треков ← нет в промтах, не добавляем
```
