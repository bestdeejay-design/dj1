# 🚀 БЫСТРЫЙ СТАРТ

## 📖 ЧТО ЭТО ЗА ПРОЕКТ

**DJ1.RU Producer Gallery** — веб-приложение для просмотра и прослушивания музыкальных треков от BEST.

**Технологии:**
- Frontend: Vanilla JS, HTML5, CSS3
- API: REST (https://api.dj1.ru)
- Данные: JSON (локально) + API (удалённо)
- Без фреймворков, без сборщиков

---

## 🎯 ТРИ ОСНОВНЫХ РЕЖИМА

### 1. Albums View
**Что показывает:** Альбомы (плейлисты) с обложками  
**Источник данных:** API `/playlists`  
**Пагинация:** Infinite scroll (Observer в body)  
**Особенности:** Ленивая загрузка треков внутри альбома

### 2. Top Tracks View ⭐
**Что показывает:** Топ треков по популярности  
**Источник данных:** API `/tracks`  
**Пагинация:** Ручная (без Observer, браузер скроллит)  
**Особенности:** Эталонная реализация пагинации ✅

### 3. By Tags View 🏷️
**Что показывает:** Треки по жанрам/настроению/вокалу  
**Источник данных:** JSON локально (8202 трека)  
**Пагинация:** Infinite scroll (IntersectionObserver)  
**Особенности:** Мгновенная загрузка (данные в памяти)

## 🐛 БЫСТРАЯ ДИАГНОСТИКА ПРОБЛЕМ

### Проблема: Не подгружаются треки после 20 штук
**Где смотреть:** `script.js` строка ~1948 (`loadMoreTagTracks`)  
**Что проверять:**
1. Флаг `isLoadingTagTracks` — не застрял ли?
2. Флаг `tagTracksHasMore` — не стал ли false раньше времени?
3. IntersectionObserver — срабатывает ли?

**Как отладить:**
```javascript
// Вставить в начало loadMoreTagTracks()
console.log('Load called:', { tagTracksPage, tagTracksHasMore });
```

### Проблема: Трек играет но нет подсветки в списке
**Где смотреть:** `script.js` строка ~2040 (`updateTagTrackHighlight`)  
**Что проверять:**
1. `currentAlbum.id === 'tag-tracks'` — совпадает?
2. `currentTrackIndex` — правильный индекс?
3. Контейнер найден — `document.getElementById('tagTracksListContainer')`

### Проблема: Переключение видов ломает состояние
**Где смотреть:** `script.js` строка ~1240 (`switchView`)  
**Что проверять:**
1. Сохранение состояния перед переключением
2. Очистка таймеров и observer
3. Восстановление после возврата к виду
