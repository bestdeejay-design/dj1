## 🐞 ЧТО ИСКАТЬ (Known Issues)

Возможные проблемы которые могут проявиться:

1. **Двоение подсветки** — два трека подсвечены одновременно
   - Решение: Проверить `updateTopTrackHighlight()` убирает ли старую подсветку

2. **Скролл не работает** — трек меняется но скролла нет
   - Решение: Проверить вызов `scrollToCurrentTopTrack()` в `nextTrack()`

3. **Подсветка не там** — подсвечивается не тот трек
   - Решение: Проверить сравнение по ID в `renderTopTracks()`

4. **Бесконечный скролл ломает подсветку**
   - Решение: Проверить `setupTagTracksInfiniteScroll()` не пересоздаёт ли observer
